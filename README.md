Docker Wordpress Example
=====================

A Wordpress example that can deploy on Docker Datacenter through the UI.

DDC and Docker 1.11
-------------------
UI Deploy:

Copy and paste contents of docker-compose.yml

CLI deploy:
```
# Source client bundle
docker-compose up
```

DDC and Docker 1.12
-------------------
CLI deploy:
```
# Source client bundle
./run-wordpress-service.sh
```
